Good First Issues empowers first-time contributors of open-source software

Amidst the ongoing pandemic, I decided to use my free time to learn GitHub's GraphQL API, and put together a simple website to aggregate and display GitHub issues labelled as "Good First Issue".
This website is primarily targeted at developers who want to contribute to open source software but do not know where or how to start. By providing developers with filters, this website empowers them to browse and select issues and repositories based on programming languages that they are comfortable with. In addition, they can select the type of issues they want to address.
